# Documentação da arquitetura do React Native 

A arquitetura utilizado nos meus projetos React e React native geralmente seguem essa estrutura:

|-- App  <br>
|   |-- Handles  <br>
|   |-- Services  <br>
|   |-- Config  <br>
|-- Models  <br>
|-- Routes  <br>
|-- Utils  <br>
|-- Components  <br>
|-- Views  <br>
